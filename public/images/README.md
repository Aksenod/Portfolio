# Инструкция по использованию изображений

## Вариант 1: Простое копирование (без обработки)

Если хотите использовать изображения в исходном формате:

1. **Поместите изображения** (jpg, png, webp и т.д.) в папку `public/images/raw/`
   - Нужно минимум 5 изображений
   - Имена файлов могут быть любыми

2. **Запустите скрипт копирования:**
   ```bash
   npm run copy-images
   ```

3. **Скрипт скопирует файлы** в `public/images/` с именами `photo-1.jpg`, `photo-2.png` и т.д. (в исходном формате)

4. **Обновите пути в `app/page.tsx`** в соответствии с форматом ваших файлов

## Вариант 2: С обработкой и сжатием

Если хотите оптимизировать изображения:

1. **Поместите исходные изображения** в папку `public/images/raw/`

2. **Запустите скрипт обработки:**
   ```bash
   npm run process-images
   ```

3. **Скрипт автоматически:**
   - Сожмёт изображения до максимум 1200px (сохраняя пропорции)
   - Сохранит в исходном формате (jpg останется jpg, png останется png) с оптимизацией
   - Сохранит результаты в `public/images/` с именами `photo-1.jpg`, `photo-2.png` и т.д.

4. **Обновите пути в `app/page.tsx`** в соответствии с форматом ваших файлов

## Проверка формата файлов

После копирования/обработки запустите:
```bash
node scripts/get-image-paths.js
```

Это покажет точные пути к файлам для использования в коде.

## Примечания:
- Скрипты обрабатывают изображения в алфавитном порядке
- Если изображений больше 5, будут использованы первые 5
- Если изображений меньше 5, добавьте недостающие
